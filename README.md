# KlikRasa ML Model

Model ini digunakan untuk:
- Retrieval of culinary cultural knowledge (RAG)
- Summarization
- Storytelling (retelling sejarah/makna)
- Chat-based Q&A based on FAISS index

LLM: Llama-3.1-8B-Instruct (Ollama)
Embedding: all-MiniLM-L6-v2
Dataset: kue tradisional nusantara (100+ entries)
